# OZON FBO 发货完整流程指南

> **FBO (Fulfillment by Operator)** = OZON 官方代发模式。卖家把货发到 OZON 仓库，OZON 负责仓储、打包、配送、客服、退换货全流程。
>
> 优势：前台显示"OZON配送"，转化率高；物流时效快（1-3天）；平台算法倾向FBO商品，搜索排名靠前。
>
> 适用场景：**爆款、高周转品、小件轻货**（如斜挎包/腰包，非常合适FBO）。

---

## 一、FBO 与 FBS 对比（了解全貌）

| 维度 | FBO | FBS |
|------|-----|-----|
| 仓储 | OZON 仓库 | 你自己的仓库/海外仓 |
| 打包发货 | OZON 负责 | 你负责 |
| 配送时效 | 1-3天 | 3-7天+ |
| 前台标识 | "OZON"配送 | 显示卖家配送 |
| 搜索权重 | 高 | 中 |
| 退货处理 | OZON 处理 | 你处理 |
| 仓储成本 | 有（按体积/天数） | 无 |
| 操作复杂度 | 低（发货后不用管） | 高（每天处理订单） |

## 二、FBO 完整发货流程（7步）

### 第1步：确认商品符合FBO要求

**商品要求：**
- 非禁运品类（绝大多数实物商品均可）
- 商品必须有完整包装（不影响二次销售）
- 非液体/危险品/易腐品（这些只能用FBS）
- 无侵权/仿牌风险

**包装要求：**
- 每个商品须有独立包装（气泡袋/塑料袋）
- 外箱须是标准瓦楞纸箱（无破损）
- 单箱重量不超过 25kg
- 单箱体积不超过 0.2m³
- 包裹尺寸任一边不超过 150cm

### 第2步：登录卖家后台 创建新供货

1. 登录 seller.ozon.ru
2. 导航到 **Логистика → FBO → Поставки**（物流 → FBO → 供货）
3. 点击 **+ Создать поставку**（创建新供货）
4. 选择供货类型：**Новая поставка**（新商品首次入仓）或 **Дополнительная поставка**（补货）
5. 勾选要发货的商品（如你的斜挎包 SKU）
6. 填写预计发货数量，点击 Далее

### 第3步：选择 OZON 仓库

OZON 在俄罗斯有多个仓库。主要仓库及推荐：

| 仓库名称 | 城市 | 推荐 |
|---------|------|------|
| **Софьино (Sofino)** | 莫斯科州 | 首选 |
| **Коледино (Koledino)** | 莫斯科州 | 热销品 |
| **Подольск (Podolsk)** | 莫斯科州 | 次选 |
| **Екатеринбург (Exb)** | 叶卡捷琳堡 | 乌拉尔覆盖 |
| **Новосибирск** | 新西伯利亚 | 西伯利亚覆盖 |
| **Казань** | 喀山 | 伏尔加区 |

**新手首发建议选 Софьино**，该系统最成熟，入库效率最高。

### 第4步：打印标签和装箱单

**商品标签：**
- 后台 Поставки 页面下载 штрихкоды（条码）
- 标签规格 76x40mm，推荐用 60x40mm 热敏纸
- **每个单品贴一个标签**（100个货=100个标签）
- 贴在商品外包装上，不可折叠，不可覆盖商品原有条码

**外箱标签：**
- 每个外箱需要箱标（Этикетки для коробов）
- 贴在箱子外侧右上角

**装箱单：**
- 下载 Упаковочный лист，打印2份
- 一份放箱内，一份给物流司机

### 第5步：安排送货到 OZON 仓库

**送货方式选择：**
- **Курьер OZON**（OZON合作快递上门取件）→ 中小批量推荐
- **Самостоятельная доставка**（自行送货）→ 大批量
- **Транспортная компания**（第三方物流）→ 远距离

**重要提醒：**
- 务必在截止入库日期前送达
- 外箱用透明或无色胶带，不要用黄色胶带
- 外箱破损会被拒收

### 第6步：仓库验收

OZON 仓库收到货后 1-7 个工作日完成验收：

| 状态 | 含义 |
|------|------|
| Создана | 已创建，未发货 |
| В пути | 运输中 |
| На складе Озон | 已到仓，等验收 |
| В сортировке | 正在分拣验收 |
| Принята | 验收完成 |
| Частично принята | 部分验收 |
| Отклонена | 全部拒收 |

**验收差异处理：** 数量不符部分会退回或销毁（收费）。建议首批多发 3-5% 余量。

### 第7步：销售与售后

1. 验收合格商品自动进入在售状态
2. 前台显示 "OZON配送" 标识
3. 买家下单后 OZON 自动从最近仓库发货
4. 你不需要做任何操作

**FBO费用结构：**

| 费用项 | 说明 | 估算 |
|--------|------|------|
| 仓储费 | 0.01-0.3 ₽/件/天 | 小件忽略不计 |
| 发货操作费 | 15-50 ₽/件 | 按尺寸重量 |
| 最后一公里配送 | 按区域 | 约 30-60 ₽ |
| 平台佣金 | 5-15% | 斜挎包约8-12% |
| 退货处理费 | 20-50 ₽/件 | 仅退货运单计费 |

你的斜挎包（220g）属于小件轻货，FBO 总费用约 **50-80 ₽/单**，非常适合 FBO。

**退货处理：**
- 买家退货到 OZON 仓库
- 无瑕疵重新上架，有瑕疵退回你的仓库或销毁
- 斜挎包正常退货率约 3-8%

## 三、FBO 时间线

第1步 商品准备 → 1-3天
第2步 创建供货单 → 30分钟
第3步 选仓库 → 10分钟
第4步 打标签 → 1天
第5步 安排物流 → 1-3天
第6步 仓库验收 → 1-7天
第7步 上架销售 → 验收后立即上架

**合计：约 7-14天 从准备到上架**

## 四、新手避坑指南

| 常见错误 | 后果 | 解决 |
|---------|------|------|
| 标签折叠污损 | 无法扫描，商品被拒收 | 热敏不干胶，保持平整 |
| 外箱超重25kg | 仓库拒收 | 拆分成多箱 |
| 标签贴外箱上 | 找不到对应商品 | 贴每个单品包装上 |
| 过期入库 | 需重新预约 | 提前3天约物流 |
| 包装太薄 | 运输破损 | 用气泡袋 |

**建议：**
1. 首次FBO发 50-100件 测试流程
2. 先发到 Софьино 熟悉流程
3. 发货前拍照留底，供后续差异投诉
4. FBO库存低于10件时及时补货
5. 爆款走FBO，新品走FBS测试

> **总结：** FBO 是把货送到 OZON 仓库后"躺赚"的模式。前期准备工作一次性做好，后面每天只需看下销售数据。你的斜挎包（轻、小、非易碎）非常适合 FBO，建议首批发 100 件到 Софьино 仓库测试。
