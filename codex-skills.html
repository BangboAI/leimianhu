⭐ 电商运营核心（必装推荐）

6

以下 Skill 是电商运营者使用 Codex 的最高频工具，建议优先安装。

Skill Creator / Skill-Creator 必装
打造专属工作流，将重复操作封装成自定义 Skill，是构建个人效率体系的第一步。
电商场景：把"每周Listing优化流程"封装成 Skill，一键执行
find-skills 必装
自动拆解关键词，在 GitHub 搜索优质 Skill 库，快速发现适合自己业务的工具。
电商场景：搜索"Amazon PPC"、"Listing optimizer"等关键词找专用工具
Superpowers 推荐
将开发/运营流程转化为问答形式，小白也能操作的友好模式，降低使用门槛。
电商场景：让团队非技术人员也能用 Codex 检查广告数据异常
Baoyu / baoyu-skills 推荐
自动分析文章/竞品结构，生成匹配的视觉内容，非常适合内容营销。
电商场景：分析竞品详情页结构，自动生成优化版的页面视觉建议
Context Engineering AI
记忆管理，优化上下文窗口使用效率，避免会话过长导致模型拒绝响应。
电商场景：长时间分析多店铺数据时保持上下文不超限
Planning with Files 免费
将任务进度保存到文件，支持跨会话继续工作，适合长周期运营规划。
电商场景：分步骤执行选品计划，每次会话接续上次的进度

🛠️ 如何安装这些 Skill？

1
方法一（推荐）：在 Codex 对话框中输入 /skills 打开 Skill 管理面板，搜索 Skill 名称一键安装。
2
方法二：访问 GitHub.com，搜索 Skill 名称（如 HyperFrames codex skill），按 README 配置到对应目录。
3
方法三：使用 find-skills Skill（已在上方推荐），让 Codex 自动帮你搜索和安装。
💡 提示：优先安装「⭐ 电商运营核心」中的 6 个 Skill，再根据业务需求添加垂直专用 Skill。
